Лабораторна робота №5 — Реалізація симетричного шифрування електронних повідомлень
Мета роботи

Реалізувати простий алгоритм шифрування електронних повідомлень для демонстрації принципів захищеної комунікації, включаючи генерацію ключа на основі персональних даних користувача, шифрування та розшифрування повідомлень, а також демонстрацію процесу безпечного обміну.

Використані технології

Мова програмування: Python 3
Бібліотека: cryptography (модуль Fernet)
Хеш-функція: SHA-256
Кодування ключа: URL-safe Base64

Опис реалізації

У межах лабораторної роботи розроблено програму, яка реалізує симетричне шифрування текстових повідомлень за допомогою алгоритму Fernet. Ключ шифрування генерується на основі персонального рядка користувача (наприклад, комбінації імені та року народження).

Для формування ключа використовується хеш-функція SHA-256, яка створює 32-байтовий хеш від введених персональних даних. Отриманий хеш перетворюється у формат URL-safe Base64, що відповідає вимогам Fernet.

Програма реалізує:
– шифрування текстового повідомлення;
– розшифрування зашифрованого повідомлення;
– демонстрацію обміну між відправником та отримувачем;
– перевірку коректності ключа;
– обробку помилки при спробі розшифрування з неправильним ключем.

У процесі демонстрації відправник генерує ключ із персональних даних, шифрує повідомлення та передає його отримувачу. Отримувач, використовуючи ті самі дані для генерації ключа, успішно розшифровує повідомлення. У випадку використання неправильного ключа виникає виняток, що підтверджує неможливість доступу до даних без відповідного ключа.

Виконання функціональних вимог

У роботі реалізовано всі обов’язкові вимоги:
– шифрування тексту повідомлень симетричним алгоритмом;
– розшифрування отриманих зашифрованих повідомлень;
– генерація ключів на основі персональних даних користувачів;
– демонстрація процесу безпечного обміну повідомленнями.

Інструкція з запуску

Встановити бібліотеку:
pip install cryptography

Запустити програму:
python lr5_email_encryptor.py

Очікуваний результат

У консолі відображається:
– оригінальний текст повідомлення;
– зашифрований токен Fernet;
– результат успішного розшифрування;
– повідомлення про помилку при використанні неправильного ключа.

Висновок

У ході виконання лабораторної роботи реалізовано базову модель симетричного шифрування електронних повідомлень. Програма демонструє принципи генерації ключа, шифрування та розшифрування даних, а також підтверджує, що без правильного ключа доступ до зашифрованої інформації неможливий. Реалізація має навчальний характер і призначена для демонстрації базових принципів захищеної комунікації.
