# ЛР4 — Спрощена система цифрових підписів (SHA-256 + XOR)

## Опис
Проєкт демонструє принципи цифрового підпису у спрощеному вигляді:
1) створення SHA-256 хешу документа;
2) “підписання” хешу приватним ключем (XOR);
3) збереження підпису в окремому файлі;
4) перевірка підпису та виявлення змін у документі.

> ⚠️ Це навчальна демонстрація. XOR-підпис не є криптографічно стійким і не використовується в реальних системах.

---

## Функціональність (відповідає ТЗ)
- Генерація пари ключів на основі персональних даних:
  - приватний ключ = SHA-256(прізвище + дата + секрет) % 1_000_000
  - публічний ключ = (private_key * 7) % 1_000_007
- Створення хешу документа SHA-256 (по файлу)
- Підписання: signature = hash_int XOR private_key
- Перевірка: (signature XOR private_key) == hash_int
- Демонстрація підробок:
  - зміна документа → підпис стає недійсним
  - зміна підпису → перевірка не проходить
