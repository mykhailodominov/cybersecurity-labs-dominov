# Лабораторна робота: Демонстрація SQL-ін'єкцій та методів захисту

**Дисципліна:** Захист інформації / Веб-програмування / Кібербезпека  
**Тема:** SQL-ін'єкція (SQL Injection) — уразливість та захист  
**Мета:** Створити демонстраційний додаток, який ілюструє вразливість до SQL-ін'єкцій та показує, як її усунути за допомогою параметризованих запитів.

## Опис проєкту

Це консольний додаток на Python, який демонструє дві версії функціоналу:
- **Уразлива версія** — використовує пряму конкатенацію рядків у SQL-запитах → дозволяє виконувати SQL-ін'єкції.
- **Захищена версія** — використовує підготовлені запити (prepared statements) з плейсхолдерами `?` → повністю блокує ін'єкції.

Функціонал включає:
- Пошук студентів за частиною імені
- Авторизація за email та паролем

У кожному режимі показано:
- текст SQL-запиту, що виконується
- параметри (якщо є)
- результати запиту
- успішну атаку (витік даних або обхід авторизації) у вразливій версії
- повну відсутність вразливості у захищеній версії

## Використані технології

- Python 3.8+
- SQLite3 (вбудована база даних, файл `students_demo.db`)
- ANSI-кольори для терміналу (для зручного читання)
