# Лабораторна робота №1 — Аналіз безпеки паролів з використанням персональних даних

Дисципліна: **Захист інформації**

## Мета
Створити програму для аналізу безпеки паролів з урахуванням персональних даних користувача:
- виявити зв’язок між паролем і персональними даними (ім’я, прізвище, логін, дата народження тощо);
- оцінити складність пароля за критеріями (довжина, різноманітність символів, словникові/поширені слова, послідовності, повтори);
- надати конкретні рекомендації для покращення безпеки;
- протестувати на реальних персональних даних студента.

## Функціональність програми
Програма:
1. Приймає введення:
   - пароль;
   - ім’я;
   - прізвище (опційно, але рекомендовано);
   - логін/нікнейм (опційно);
   - дата народження (формат `ДД.ММ.РРРР`);
   - додатково: email / телефон / місто (опційно).
2. Перевіряє:
   - чи містить пароль ім’я/прізвище/логін (у різних регістрах);
   - чи містить пароль рік, день, місяць народження, або комбінації типу `DDMM`, `MMYYYY` тощо;
   - довжину пароля;
   - наявність різних класів символів: нижній/верхній регістр, цифри, спецсимволи;
   - повтори символів, прості послідовності (`1234`, `abcd`, `qwerty`), “слабкі” слова (мінімальний словник поширених паролів).
3. Формує звіт:
   - знайдені збіги з персональними даними;
   - виявлені слабкі місця;
   - оцінка **1–10**;
   - рекомендації та приклади, як покращити пароль.

## Вимоги
- Python 3.10+ (підійде і 3.8+, але рекомендовано 3.10+)
- Додаткові бібліотеки **не потрібні** (тільки стандартна бібліотека).
