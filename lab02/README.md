Лабораторна робота №2
Тема: Порівняльний аналіз класичних шифрів (Цезаря та Віженера)

Мета роботи:
Метою лабораторної роботи є реалізація класичних алгоритмів шифрування – шифру Цезаря та шифру Віженера – для української абетки, генерація ключів на основі персональних даних користувача, виконання шифрування та дешифрування тексту, а також проведення порівняльного аналізу отриманих результатів.

У роботі використовується повна українська абетка з 33 літер:
АБВГҐДЕЄЖЗИІЇЙКЛМНОПРСТУФХЦЧШЩЬЮЯ
Усі обчислення виконуються за модулем 33, що відповідає кількості літер алфавіту.

Генерація ключів:
Ключ для шифру Цезаря формується на основі дати народження користувача. З дати беруться лише цифри, після чого обчислюється їх сума. Отримане число нормалізується за модулем 33:
shift = (сума цифр дати) mod 33.
Наприклад, для дати 15.03.2003 сума цифр дорівнює 1 + 5 + 0 + 3 + 2 + 0 + 0 + 3 = 14.
Оскільки 14 < 33, зсув дорівнює 14.

Ключ для шифру Віженера – це прізвище користувача, введене українськими літерами. Програма перевіряє, щоб ключ містив лише символи української абетки.

Опис алгоритмів:
Шифр Цезаря є моноалфавітною підстановкою. Кожна літера відкритого тексту замінюється літерою, зміщеною на фіксовану кількість позицій у межах алфавіту.
Шифрування виконується за формулою:
E(x) = (x + shift) mod 33.
Дешифрування:
D(x) = (x − shift) mod 33.
Регістр літер зберігається, пробіли та розділові знаки не змінюються.

Шифр Віженера є поліалфавітною підстановкою. Для кожної літери тексту використовується відповідна літера ключа. Ключ циклічно повторюється.
Шифрування:
E(x_i) = (x_i + k_i) mod 33.
Дешифрування:
D(x_i) = (x_i − k_i) mod 33,
де x_i – позиція літери тексту, k_i – позиція літери ключа в алфавіті.

Реалізований функціонал програми:
Програма запитує у користувача ім’я, прізвище (ключ Віженера) та дату народження. На основі введеної дати автоматично обчислюється зсув для шифру Цезаря. Після цього виконується шифрування та дешифрування тестового тексту. Додатково програма обчислює довжину шифротексту, кількість унікальних символів (без пробілів і тире) та частку літер алфавіту серед символів шифротексту. Наприкінці виводиться порівняльна таблиця та формуються загальні висновки.

Вимоги до запуску:
Для роботи програми необхідно встановити Python версії 3.9 або вище. Додаткові бібліотеки не використовуються. Запуск здійснюється командою:
python lr2_full.py

Порівняльний аналіз:
Шифр Цезаря є простим у реалізації та має лише один параметр – зсув. Проте його криптостійкість є низькою, оскільки можливий повний перебір усіх 33 варіантів зсуву. Крім того, частоти літер зберігаються, що дозволяє застосовувати частотний аналіз.

Шифр Віженера використовує змінний зсув, який залежить від ключа. Завдяки цьому розподіл частот у шифротексті є більш рівномірним, а стійкість значно вища, ніж у шифру Цезаря. Проте для коротких або передбачуваних ключів можливі криптоаналітичні атаки.

Загальний висновок:
У ході лабораторної роботи реалізовано два класичних алгоритми шифрування та проведено їх порівняльний аналіз. Встановлено, що шифр Віженера забезпечує вищий рівень захисту порівняно з шифром Цезаря. Водночас обидва алгоритми є історичними та використовуються переважно в навчальних цілях, оскільки не забезпечують достатнього рівня безпеки в сучасних інформаційних системах.
